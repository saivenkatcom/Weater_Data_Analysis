## Installation
To run this project, you need to have the following dependencies installed:
- Python 3.11.4
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

You can install the dependencies using the following command:

python -m pip install pandas numpy matplotlib seaborn jupyter
